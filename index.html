Generative Block Game

This is a generative Game Boy x Tetris style project I coded with p5.js to sharpen my JavaScript skills. The canvas and all internal elements are rendered with JavaScript. Each block drop sequence is randomly generated (shapes, colors and positioning). I also implemented more advanced techniques such as collision detection. Once a block reaches the top of the screen container, the logic determines the game is now over, clears the screen and all variables, displays the game over text for 5 seconds and then loops. It's not intended to be playable; just fun to watch. My GitHub
